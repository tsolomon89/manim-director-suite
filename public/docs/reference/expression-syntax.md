# Expression Syntax

Mathematical expression syntax reference.

## Operators

- `+`, `-`, `*`, `/` - Arithmetic
- `^` - Exponentiation
- `%` - Modulo

## Functions

- `sin(x)`, `cos(x)`, `tan(x)` - Trigonometry
- `sqrt(x)`, `abs(x)`, `log(x)` - Common math
- `exp(x)`, `ln(x)` - Exponential/logarithm

## Constants

- `pi` or `π` - Pi (3.14159...)
- `e` - Euler's number (2.71828...)
- `tau` or `τ` - Tau (2π)

Type `\alpha` → `α`, `\beta` → `β`, etc.
