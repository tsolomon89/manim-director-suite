# File Formats

Documentation for supported file formats.

## Project Files (.pkstudio)

JSON format containing:
- Parameters
- Functions
- Keyframes
- Scene configuration
- Camera state

## Desmos JSON

Import Desmos graph state:
- Numeric parameters only (MVP)
- Viewport bounds
- Expression list

## Export Formats

- **PNG**: Raster images
- **Manim**: Python scripts
